在这里，我们研究的React.js版本为v0.13.0

下面把研究的大致流程介绍一下

1.对目录结构的设计理解

2.对所有模块划分的宏观理解

3.模块之间的协作图完善出来

4.根据源码目录给出每份代码的相应研究理解(接口设计目的，主要起到的作用，核心算法，策略，机制研究)

5.持续完善，修改....
